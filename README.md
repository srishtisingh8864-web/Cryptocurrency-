# Cryptocurrency-
This project is an interactive Cryptocurrency Dashboard built using Power BI to analyze market trends, price movements, and trading volumes of multiple cryptocurrencies.  
📌 Project Description

This project is an interactive Cryptocurrency Dashboard developed using Power BI to analyze and visualize the performance of multiple cryptocurrencies over time.

It transforms raw market data into meaningful insights by tracking price trends, trading volume, and performance metrics, enabling users to compare different cryptocurrencies and identify market patterns effectively.

🎯 Project Purpose
To analyze cryptocurrency market behavior
To track price fluctuations over time
To compare performance across multiple coins
To identify top and underperforming assets
To support data-driven decision making
🛠️ Tech Stack
Power BI (Dashboard & Visualization)
Power Query (Data Cleaning & Transformation)
DAX (Calculated Measures & KPIs)
📂 Data Source
Public cryptocurrency dataset (CSV format)
Contains:
Cryptocurrency symbols
Date-wise price data (Close, High, Low)
Trading volume
📊 Key Features
Interactive dashboard with multiple pages
Time-based filtering using slicers
Close Price, High, Low tracking
30-Day Moving Average analysis
Average Daily Volume visualization
Best & Worst performers (30 & 365 days)
Multi-cryptocurrency comparison table
⭐ Project Highlights
Built a multi-page analytical dashboard for better user experience
Performed trend analysis using moving averages
Designed comparative analysis across 15+ cryptocurrencies
Implemented performance tracking (short-term & long-term)
Created clean and professional dark-themed UI
Delivered insights using data storytelling approach
📈 Business Insights
High volatility observed in certain cryptocurrencies during peak periods
Stable coins showed minimal price variation
Trading volume spikes align with major price movements
Some assets outperform in short-term but not in long-term
🧩 Tasks Performed
Data cleaning and preprocessing using Power Query
Created calculated measures using DAX
Designed interactive visuals and KPIs
Built filters and slicers for user interaction
Analyzed trends and performance metrics
Structured dashboard for storytelling
